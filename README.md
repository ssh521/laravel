# laravel
 라라벨 시작하기

## Install

```
laravel new laravel
```
```
cd laravel
```
```
code .
```
```
composer install
```
```
npm install
```
```
php artisan key:generate
```
```
npm run build
```
```
php artisan serve
```

## DB setting

mysql db 생성하기  
  
.env 수정

    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=laraveldb
    DB_USERNAME=laraveldb
    DB_PASSWORD=*******

```
php artisan migrate
```
